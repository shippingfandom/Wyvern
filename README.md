# Wyvern
A programming language written in [MiniScript](https://github.com/JoeStrout/miniscript)

# Handbook
The handbook is in the `notes` folder

# Installation
Two ways to install the **wyvic** compiler:
- Manual (Hand copy from [src](src))
- Greybel-JS (`greybel build src/wyvic/wyvic.ms -id /root -ac -acp -ci` from the root of the project)
